# GoOut

Rails app using Devise. Allows to create events and participations of users.

## Usage

```ruby
bundle install
rails db:migrate
rails db:seed
```

After that, you will be able to create an account, connect, create events, and disconnect (ect...)

## Versions used

ruby 3.4.2
rails 8.0.3

## Necessary gems
dotenv-rails
faker
devise
